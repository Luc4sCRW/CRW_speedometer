# ***GERMAN***
Ein simples GTA V HUD-Script, das die aktuelle **Geschwindigkeit**, den **Gang**, die **RPM** und den **Tankstand** anzeigt.

# 🔧 Features

- Geschwindigkeit in **KMH** oder **MPH**
- **Gang Anzeige** mit Neutral (N) & Rückwärts (R)
- **RPM-Balken**
- Tank Anzeige
- Motor ein-/ausschalten (Standard Taste: M)
- **Zwei Styles**: *glass* und *clean*
- **Zwei Formen**: *rounded* und *squared*
- Ein-/Ausblenden bei GTA MAP Menu und Motor aus
- Einfach anpassbar (zum Beispiel in Config.lua)

# 🚀 Installation

1. Lege den Ordner `CRW_speedometer` in deinen `resources/` Ordner.
2. Füge folgendes in deine `server.cfg`:
 `            ensure CRW_speedometer           `
3. Passe die *config.lua* nach deinen Wünschen an (Einheiten, Style, Form, Update-Tick).


# ***ENGLISH***
A simple GTA V HUD script that displays the current **speed**, **gear**, **RPM**, and **fuel level**.

# 🔧 Features

- Speed in **KMH** or **MPH**
- **Gear indicator** with Neutral (N) & Reverse (R)
- **RPM bar**
- Fuel gauge
- Turn engine on/off (default key: M)
- **Two styles**: *glass* and *clean*
- **Two shapes**: *rounded* and *squared*
- Hide/show in GTA MAP menu and when engine is off
- Easily customizable (for example, in Config.lua)

# 🚀 Installation

1. Place the `CRW_speedometer` folder in your `resources/` folder.
2. Add the following to your `server.cfg`:
 `            ensure CRW_speedometer           `
3. Customize *config.lua* to your liking (units, style, shape, update tick).
